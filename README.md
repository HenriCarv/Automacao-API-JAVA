# Automacao-API-JAVA
Automação de API em JAVA

## Pré-requisitos
- Java 11 ou superior
- Maven 3.6.3 ou superior

## Versões
- Java: 11
- Maven: 3.6.3

## Dependências
- JUnit 5
- RestAssured
- Maven Surefire Plugin

## Como rodar um teste
1. Clone o repositório:
   ```bash
   git clone https://github.com/HenriCarv/Automacao-API-JAVA.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd Automacao-API-JAVA
   ```
3. Execute os testes usando Maven:
   ```bash
   mvn test
   ```
